Our first meetup was in November 2012, come and celebrate 1 year of Node.ninjas!

This month, we're looking at using Node.js to talk to hardware as well as using Node to run your platform as a service.

We've got Dominik talking about building underwater robots using OpenROV and David talking about Field Marshall, an open source PaaS. Hugh will also be bringing in his Node theramin to show off, so it looks to be an exciting month!
